Sleepyowl's Zoo Website

This is application built with React framework. It creates an online interactive zoo where users can filter, sort, like and unlike the animals. The user could learn about the animals' type, geo range, active time, and lifespan information.

Visit webpage → Github Repo →

Functionality

  • Sort: The user can sort the animals based on their lifespan through a dropdown menu. Only one type of sorting method could be applied at a time.
  • Filter: The user can filter the animals based on active time and animal type. When none or all of the filters are selected, all the animals would be displayed. Within each filter type, the filter applies OR operation. Between the two types, the filter applies AND operation.
  • Like: The user can like and unlike any animal. An aggregation section would show the average lifespan of all the liked animals.
  • Reset: the user could reset either by clicking the "reset" button, or undoing the liking, sorting and filtering operations.

Components and Data

Each animal is represented by an "animal card" component. The animal data is represented by a Json data file. The data is passed down to the component using prop that include Json data, liked status, and function for aggregation.

The rendering process contains the following steps:

  1. Filter the animal data based on the user input conditions.
  2. Map the eligible animals to "animal card component using props.
  3. Sort the animal cards based on the user chosen method
  4. Update the change of like / unlike stuts for each animal. Display the liked animal list and calculate their average lifespan.

States

State Variables

The program uses the following variables to keep track of the data and user input:

  • Animal Data to display
  • User liked list
  • Aggregated lifespan information of liked animals
  • Applied filter types
  • Applied sorting method

State Change

There are there major user inputs that triggers state change:

  • Like/Unlike button: changes the users' liked list and average favorite animal lifespan aggregation.
  • Dropdown menu (sorting method): changes the sorting method of animals based on lifespan.
  • Checkbox (filtering conditions): controls each animal component's display based on given condition.
